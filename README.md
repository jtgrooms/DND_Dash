# R Shiny D&D Dashboard 
Code to build an R Shiny Dashboard using 5th Edition Dungeons and Dragons Character data

### app.R
Code to build the R Shiny Dashboard using rshiny, rshinydashboard, highcharter, plotly, and hmisc

### deploy_app.R
Deploys the shiny app to shinyapps.io. This code assumes you have already gone through the steps to set up your shinyapps.io account using the following instructions
- https://shiny.posit.co/r/articles/share/shinyapps/
